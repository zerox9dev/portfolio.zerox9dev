# Project Content

Projects live here as MDX case studies, split by locale:

- `en/<slug>.mdx`
- `ru/<slug>.mdx`
- `ua/<slug>.mdx`

Each file uses frontmatter:

```mdx
---
title: Example title
slug: example-slug
strapline: Short project summary
category: Design
link: https://example.com
tech:
  - Next.js
  - Supabase
logo:
  asset:
    url: /images/logo.ico
media:
  - asset:
      url: /cover.png
    description: Cover image
---
```
