## Задача

Я присоединился к команде FreelanceHunt, крупнейшей фриланс-платформы Украины с 15-летней историей. Основатели решили полностью обновить сервис, которым каждый месяц пользуется миллион человек. Старый интерфейс не соответствовал современным стандартам, пользователи жаловались на сложность, а метрики показывали проблемы с конверсией.

## Решение

Я тесно работал с Product Manager над полным редизайном платформы. Начал с глубокого пользовательского исследования — анализировал поведение через Hotjar, работал с GA4 и Amplitude, чтобы понять проблемные места.

С нуля создал Design System — полноценную библиотеку компонентов для веб- и мобильной версии. Для такого большого проекта это было критично, чтобы команда разработки могла параллельно работать над разными частями продукта.

Провёл исследование продуктовых метрик, чтобы каждое дизайн-решение было связано с бизнес-целями. Вёл проект от ресерча до реализации вместе с разработчиками, контролируя корректное внедрение дизайна.

## Результат

Создал полный редизайн платформы с аудиторией 1M+ пользователей. Построил масштабируемую дизайн-систему, которая позволяет команде быстро развивать продукт. Завершил свою часть проекта и передал результаты команде для дальнейшей реализации.
